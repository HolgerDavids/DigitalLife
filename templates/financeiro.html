Importar Extrato
Importe seu extrato bancário em OFX ou CSV diretamente para o sistema

Como exportar o extrato do seu banco

  1. Bradesco Conta Corrente: Internet Banking → Conta Corrente → Extrato → Exportar → OFX ou CSV
  2. Bradesco Cartão de Crédito: Internet Banking → Cartões → Fatura → Exportar Fatura → PDF
  3. C6 Bank Conta: App → Extrato → Exportar → CSV ou OFX
  4. C6 Bank Cartão: App → Cartão → Fatura → Baixar Fatura → PDF
  5. Itaú: Internet Banking → Extrato → Exportar → OFX  |  Cartão → Fatura → PDF
  6. Nubank: App → Perfil → Exportar dados → CSV  |  Fatura → PDF
  7. Inter: App → Extrato → Exportar → CSV ou OFX  |  Cartão → PDF
  8. Banco do Brasil: Internet Banking → Extrato → Exportar → OFX
  9. Santander: Internet Banking → Extrato → Salvar como → OFX

🤖 Telegram Bot

Controle suas finanças pelo Telegram! Após configurar, envie comandos como /saldo, /extrato e /lc Mercado 150 Alimentação direto no chat.

Verificando status...
Obtido com @BotFather no Telegram
URL pública do seu site
Separado por vírgula — vazio = todos
Comandos disponíveis
/ajuda — lista todos os comandos
/saldo — saldo do mês
/extrato [N] — últimas transações
/relatorio [AAAA-MM] — resumo
/lc DESC VALOR CAT — lançamento rápido
/categorias — lista categorias
/busca TEXTO — pesquisa
/deletar ID — remove transação
🎙️ Voz ativada: envie um áudio dizendo por exemplo
"gastei 50 reais no mercado" ou "qual é meu saldo?"
Texto livre também funciona: basta digitar sem usar /
💡 Seu Chat ID: envie /config para o bot para descobrir seu ID e restringir o acesso.
